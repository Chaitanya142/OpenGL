To create new app do below changes
1)Copy Current Folder

2)Change package name in 
	~\app\src\main\AndroidManifest.xml
	i.e. package="com.example.window" 
	
	Folder name 
	~\app\src\main\java\com\example\window
	
	In all Java files
	~\app\src\main\java\com\example\window
	MainActivity.java	
	i.e. package com.example.window;

3)In App Name String
	~\app\src\main\res\values
	string.xml
	i.e.<string name="app_name">Window</string>

4)Write down necessary code

5)Keep internet connetion on and execute commands given in Cmd.png
